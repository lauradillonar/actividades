Actividades de AprendeJavaScript.org

    Actividad 1
  • 1.1 Programa una función que cuente el número de caracteres de una cadena de texto.
  • 1.2 Programa una función que te devuelva el texto recortado según el número de caracteres indicados.
  • 1.3 Programa una función que dada una String te devuelva un Array de textos separados por cierto caracter.
  • 1.4 Programa una función que repita un texto X veces.
    Actividad 2
  • 2.5 Programa una función que invierta las palabras de una cadena de texto.
  • 2.6 Programa una función para contar el número de veces que se repite una palabra en un texto largo.
  • 2.7 Programa una función que valide si una palabra o frase dada es un palíndromo (que se lee igual en un sentido que en otro).
  • 2.8 Programa una función que elimine cierto patrón de caracteres de un texto dado.
